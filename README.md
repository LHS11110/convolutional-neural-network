# 합성곱 신경망 구현
본 리포지토리에서는 합성곱 신경망의 원리를 파악하고, 이를 Python 언어로 구현한다.
## 합성곱 신경망의 구조
합성곱 신경망을 알려면 먼저 input이 (높이, 너비, 채널)로 주어짐을 알아야 한다. 이는 각각을 하나의 차원으로 생각해볼 수 있으며 따라서 입력은 총 3차원이 된다. <br> 높이와 너비는 픽셀 수를 의미하고 채널은 색 성분을 의미한다. 예를 들어 RGB로 색감을 표현하는 $128 \times 128$ 크기의 이미지는 높이 128, 너비 128, 채널은 3이 된다.
### 합성곱 연산
합성곱 연산에는 input과 kernal 두 개념이 필요하다. kernal은 filter라고도 불리며 input이 $n \times m \times l$이라면 홀수 $x \leq n, y \leq m$에 대해 kernal의 크기는 $x \times y$이어야 한다. <br> 즉, kernal은 채널 차원을 가지지 않으며 각 높이와 너비 차원의 길이가 input의 높이, 너비 차원의 길이보다 짧아야 한다. 여기서 커널의 높이와 너비가 홀수인 이유는 중앙을 만들어주기 위해서다. <br>
kernal은 대부분 $3 \times 3$ 또는 $5 \times 5$의 크기를 갖도록 구현한다. 합성곱 연산을 거친 후의 결과물을 특성 맵(feature map)이라고 하며, 이 또한 마찬가지로 채널 차원을 갖지 않고 높이와 너비 차원만 주어진다.
